# Yanshu Shingala — Portfolio

Hey there! 👋 I'm **Yanshu Shingala**, an AI/ML and Full-Stack Developer. This repository contains the source code for my personal portfolio website.

The site is designed with a bold, high-contrast **Bauhaus / Brutalist aesthetic** (sharp layouts, solid offset shadows, and newsprint/monospace typography). It showcases my projects, machine learning work, and front-end development capabilities.

---

## 🎨 Portfolio Features
* **Bauhaus Design System**: A custom design system built with custom solid offsets (`shadow-bauhaus`), bold layouts, and responsive grids.
* **Dual Layout Modes**: Toggle between an interactive **3D Card Slider stage** and a clean **Grid Layout** to view projects.
* **Interactive Playgrounds**: An embedded section featuring interactive mini-playgrounds.
* **Fully Responsive**: Optimized typography and spacing scales for mobile, tablet, and desktop screens.
* **Theme Support**: Integrated light/dark modes using active CSS variants.

---

## 💻 Tech Stack
* **Core**: React 18/19, TypeScript, Vite
* **Styling**: TailwindCSS v4, Custom CSS variables
* **Animations**: CSS transitions and micro-animations

## 🛠️ Run Locally

### Prerequisites
Make sure you have **Node.js** (v18+) installed on your machine.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Yanshu04/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure environment variables:**
   Duplicate the `.env.example` file and rename it to `.env.local`:
   ```bash
   cp .env.example .env.local
   ```
   Add any keys or configurations needed.

4. **Start the development server:**
   ```bash
   npm run dev
   ```
   Open your browser to `http://localhost:5173/` (or the port specified by Vite) to view the portfolio.
